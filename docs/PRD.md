# 《棋天道》—— 小说产品需求文档（PRD）

## 0. 项目概览

《棋天道》是一部古典修仙风格的长篇小说，全文 **180 章、三卷**，每卷约 **60 章**。核心卖点在于：主角以**智谋算计**型性格，通过**系统化分析方法**（炼丹 + 阵法 + 数据分析）在修仙世界中逐步攀升，最终参与改写整个世界的天道规则。

---

## Problem Statement

传统修仙小说中存在大量同质化套路（退婚流、重生复仇、系统叠加、开局无敌），读者对这类模式已产生审美疲劳。作者希望创作一部**无外挂、无 CP、古典基调、爽文节奏**的修仙小说，以独特的"系统化修仙"视角和"天道因果"的核心冲突，提供差异化阅读体验。

---

## Solution

创作一部结构完整的修仙小说，包含：

1. **180 章完整大纲**（分卷 + 分章标题 + 摘要 + 视角 + 碰撞结果标注 + 反套路说明）
2. **人物设定体系**（主角、重要配角、反派、势力关系）
3. **修炼体系设定**（境界划分、丹道体系、阵法体系）
4. **世界观设定**（天道崩解、法则碎片、秘境机制）

核心差异化设计：
- **系统化修仙**：主角靠实验记录 + 数据分析 + 变量控制炼丹，而非天赋直觉
- **孤家寡人优势**：无 CP = 无盟友 = 没有人脉压力 = 靠信息差和布局取胜
- **天道因果**：旧天道本身有缺陷，主角最终选择创造新天道而非修复旧天道

---

## User Stories

1. 作为一位修仙小说读者，我希望看到主角靠自身智谋和分析能力而非外挂升级，这样我能获得更真实的成长共鸣
2. 作为一位读者，我希望主角的性格是智谋算计型，这样我能看到精心布局的爽感而非蛮力碾压
3. 作为一位厌倦了退婚流和重生复仇的读者，我希望这部小说不包含这些同质化套路
4. 作为一位喜欢炼丹元素的读者，我希望炼丹被系统化、科学化地描写，而非仅靠天赋灵感
5. 作为一位喜欢阵法元素的读者，我希望阵法被深度融入叙事，而非仅作为战斗工具
6. 作为一位喜欢秘境探索的读者，我希望秘境有独特的世界观解释（补天碎片），而非随机刷宝
7. 作为一位古典修仙爱好者，我希望世界观包含天道、因果、劫数等古典概念
8. 作为一位追求爽文节奏的读者，我希望主角持续获得成长、打脸对手、能力远超预期
9. 作为一位无 CP 偏好的读者，我希望主角没有感情线，专注修仙主线
10. 作为一位读者，我希望每卷有独立的主题和叙事节奏，避免整本书单调
11. 作为一位读者，我希望主角的最终抉择（新天道 vs 旧天道）具有哲学深度，而非简单的正邪对立
12. 作为一位创作者，我希望大纲包含反套路说明（not-template note），确保每个章节都有明确的差异化设计
13. 作为一位读者，我希望前 3 章有强钩子，吸引读者继续读下去
14. 作为一位读者，我希望战力体系自洽，前后不矛盾
15. 作为一位读者，我希望主角的丹魂能力有清晰的规则和限制，而非万能金手指

## Implementation Decisions

### 核心设定决策

**1. 主角设定**
- 姓名：陆衡
- 出身：没落修仙世家，家族被灭
- 性格：智谋算计型，冷静到近乎冷酷，隐忍蛰伏
- 思维方式：将复杂事物拆解为可分析的模块（丹方、阵法、剑招）
- 感情线：无 CP，专注修仙
- 金手指：无外挂，靠自身天赋 + 系统分析方法

**2. 修炼体系**
- 境界划分：炼气（1-9层）→ 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 飞升
- 八层大境界，每层有若干小阶段
- 金丹期获得"分析型丹魂"（核心能力，非金手指）
- 化神期获得"法则层面感知"能力

**3. 丹道体系**
- 核心概念：灵草丹性之间存在固定比例关系（类似数学公式）
- 主角方法：实验记录 + 变量控制 + 数据分析
- 丹书：祖传的无字丹书，遇特定条件显字，内容为无名炼丹师的实验记录（非功法非丹方）

**4. 阵法体系**
- 主角方法：将阵法分解为"灵力节点（位置）+ 灵力路径（连线）+ 灵力流向（方向）"
- 独创"丹阵合一"思路：将丹方"灵草比例"概念转化为"灵力比例"

**5. 世界观**
- 天道崩解：上古之战撕裂天地法则，碎片散落在各地秘境
- 法则碎片：补天产物，不能"取走"只能"加固"，取走会加速崩解
- 秘境机制：天道修复时产生的碎片空间，内含法则碎片
- 天道有缺陷：旧天道本身已破损，无法修复，只能创造新天道

**6. 核心冲突**
- 表层：各宗门争夺法则碎片
- 中层：天道锚定候选人之争（谁来决定新天道规则）
- 深层：天道有缺陷的真相 vs 所有人的认知

**7. 三卷结构**
- 第一卷：凡尘入道（1-60章）—— 炼气 → 金丹，建立系统化修仙者人设
- 第二卷：天机棋局（61-120章）—— 元婴 → 化神，宗门博弈 + 暗战 + 天道真相
- 第三卷：天道因果（121-180章）—— 万丹炼制 + 新天道建立 + 最终抉择

## Testing Decisions

### 原创性验证（Familiarity Check）

**5a. Pitch Test — 每卷一句话 pitch：**

- **第一卷**：一个四系杂灵根的没落世家子弟，靠祖传的一本炼丹实验记录，用系统化分析方法在修仙界杀出一条路。→ **yes**（"废柴逆袭"模式存在风险）
  - **替换方案**：不强调"废柴"，强调"系统分析方法"的独特性。主角灵根虽杂但分析能力极强，他不是从底层爬升，而是用不同方法论碾压传统天才。

- **第二卷**：主角被卷入天衍宗高层的权力博弈，发现自己最敬重的师尊在暗中加速天道崩解以保全小宗门。→ **no**（独特度高）

- **第三卷**：主角发现旧天道本身有缺陷，决定用万丹之灵创造全新天道，在天下修士的反对中完成规则重写。→ **no**（独特度高）

- **全书整体**：一个没有感情线、没有外挂的修仙者，用数据分析的方法理解修仙本质，最终改写世界规则。→ **unsure**（"改写世界规则"有一定通用性，但"系统化炼丹+阵法+因果武器化"的组合是独特的）

**5b. 反模式检查**
- 多系统叠加：❌ 无
- 系统阴谋论：❌ 无（丹尊的计划是利益驱动，非 meta）
- 重生复仇堆砌：❌ 无
- 无敌流过早：❌ 无（主角逐步升级）
- 退婚流泛滥：❌ 无
- 无限流缝合：❌ 无

**5c. Name Sensitivity Check**
- "陆衡" + "修仙"：无知名作品
- "棋天道" + "小说"：无知名作品
- 其他角色名均为通用修仙风格命名，未与知名作品冲突

## Out of Scope

以下事项不在本次 PRD 范围内：

1. **正文字数**：本 PRD 仅提供大纲和设定，不包含完整章节正文的写作
2. **插图/封面设计**：不涉及视觉素材制作
3. **多平台发布策略**：不涉及番茄、起点等平台的具体发布方案
4. **系列化规划**：不涉及第二部的构思
5. **多语言翻译**：不涉及翻译版本规划
